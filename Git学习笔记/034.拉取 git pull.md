### 4.拉取 git pull

```text
git pull <远程主机名> <远程分支名>:<本地分支名>

$ git pull --allow-unrelated-histories
```

git pull操作其实是git fetch 与 git merge 两个命令的集合。 git fetch 和 git merge FETCH_HEAD 的简写。
